Ninja Game

Ninja Gameis a 2D pixel-art platformer built with Pygame, inspired by a full YouTube tutorial. You play as a ninja who can run, jump, and dash through platform-based levels while avoiding hazards and trying to reach the goal. The game focuses on smooth movement, precise controls, and classic platformer mechanics.

Game Controls

Player Controls

            Arrow Left / Right — Move left / right

            Arrow Up — Jump

            X — Dash

            ESC — Quit game

Map Editor Controls

            W / A / S / D — Move camera

            Mouse Wheel — Select block

            Shift + Mouse Wheel — Select block variant

            T — Tilemapping (merge blocks)

            O — Save map

            ESC — Quit editor

Features

            Smooth platformer movement (gravity, jumping, dashing)

            Tile-based level system

            Custom in-game map editor

            Pixel-art visual style
